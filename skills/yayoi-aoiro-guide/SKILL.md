# やよいの青色申告オンライン ガイド

## 重要: 最新情報の取得

**このスキルで回答する際は、必ずWebSearchまたはWebFetchで公式サポートサイトから最新情報を取得してから回答すること。**

本ドキュメントの情報は参考用であり、料金・機能・操作手順は変更される可能性があります。

```
必須ワークフロー:
1. ユーザーの質問内容を特定
2. WebSearchで最新情報を検索（または該当サポートページをWebFetch）
3. 取得した最新情報に基づいて回答
4. 情報源のURLを明記
```

## 概要

やよいの青色申告オンラインは、個人事業主向けのクラウド青色申告ソフトです。

**料金**:
- 初年度無料キャンペーン（開業届提出から1年間）
- 2年目以降: セルフプラン 9,680円/年、ベーシックプラン 15,180円/年

**対象**: 個人事業主（青色申告・白色申告対応）

## 主な機能

| カテゴリ | 機能 |
|---------|------|
| 日常業務 | 取引入力、銀行・カード連携、経費精算 |
| 決算・申告 | 青色申告決算書、確定申告書B、e-Tax連携 |
| その他 | レポート、固定資産台帳、消費税集計 |

## クイックスタート

### 1. 初期設定

```
手順:
1. 事業者情報の登録（屋号、開業日、決算期など）
2. 銀行・クレジットカード連携
3. 勘定科目の確認
4. 開始残高の入力
```

**サポートリンク**: [使い方動画一覧](https://www.yayoi-kk.co.jp/startupguide/aoiro_ol/)

### 2. 日常の記帳

#### かんたん取引入力

取引を「収入」「支出」「振替」の3つから選んで入力します。

```
例: 業務委託報酬の入力
1. 「収入」を選択
2. 取引日、科目（売上高）、金額、摘要を入力
3. 取引手段（普通預金など）を選択
4. 登録
```

#### スマート取引取込

銀行・カードの明細を自動取得し、AIが勘定科目を推測します。

```
手順:
1. 金融機関連携を設定
2. 自動取得された明細を確認
3. 勘定科目を確認・修正
4. 一括登録
```

**サポートリンク**: [スマート取引取込](https://support.yayoi-kk.co.jp/subcontents.html?page_id=23742&grade_id=Blue)

### 3. 確定申告の流れ

```
年間スケジュール:
1月-12月: 日常の記帳
12月末: 決算整理（減価償却、棚卸など）
1月: 青色申告決算書の作成
2-3月: 確定申告書Bの作成・e-Tax送信
```

**サポートリンク**: [確定申告、業務の流れ](https://support.yayoi-kk.co.jp/business/shinkoku/index.html)

### 4. 青色申告決算書の作成

日常の記帳が完了していれば、自動集計されます。

```
手順:
1. 「確定申告」メニューから「青色申告決算書」を選択
2. 自動集計された金額を確認
3. 必要に応じて修正
4. PDFダウンロード or e-Tax連携
```

### 5. 確定申告書Bの作成

```
手順:
1. 「確定申告書B」を選択
2. 事業所得を自動反映（青色申告決算書から）
3. 各種控除（社会保険料、生命保険料、配偶者控除など）を入力
4. 税額を自動計算
5. e-Tax送信 or PDFダウンロード
```

## よくある質問

### Q: 家事按分の設定方法は？

A: 「設定」→「家事按分」から、勘定科目ごとに按分割合を設定できます。決算時に一括按分処理が実行されます。

**サポートリンク**: WebSearchで `やよいの青色申告 オンライン 家事按分` を検索

### Q: e-Taxで確定申告する方法は？

A: マイナンバーカード方式またはID・パスワード方式で連携できます。事前にe-Taxの利用者識別番号が必要です。

**サポートリンク**: WebSearchで `やよいの青色申告 オンライン e-Tax` を検索

### Q: 開始残高の入力方法は？

A: 「設定」→「残高設定」から、事業開始日時点の現金・預金・借入金等の残高を入力します。

### Q: サポートはどこで受けられる？

A: プランにより異なります。
- セルフプラン: WebFAQ、チャットサポート
- ベーシックプラン: 上記 + 電話・メール・画面共有サポート
