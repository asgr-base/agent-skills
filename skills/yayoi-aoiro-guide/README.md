# やよいの青色申告オンライン ガイド スキル

## 概要

このスキルは、個人事業主向けのクラウド青色申告ソフト「やよいの青色申告オンライン」の利用方法を案内するためのものです。

## 使用タイミング

以下のような質問時に自動起動されます:

- 「やよい」「弥生」「青色申告」関連の質問
- 確定申告、記帳、経費入力などの個人事業主業務
- やよいの青色申告オンラインの操作方法

## 主な機能

- 日常の記帳方法の案内（かんたん取引入力、スマート取引取込）
- 確定申告の流れとスケジュール
- 青色申告決算書・確定申告書Bの作成方法
- 家事按分、減価償却などの決算整理
- e-Tax連携方法
- よくある質問への回答

## ファイル構成

- `SKILL.md`: メイン指示（500行未満、Progressive Disclosure準拠）
- `REFERENCE.md`: 詳細リファレンス（トラブルシューティング、詳細手順など）
- `README.md`: このファイル

## 重要な注意点

このスキルは**必ずWebSearchまたはWebFetchで最新情報を取得してから回答**します。

理由:
- やよいの青色申告オンラインの機能・料金は随時変更される
- 税制改正により操作手順が変わる可能性がある
- 公式サポートサイトが最も信頼性の高い情報源
