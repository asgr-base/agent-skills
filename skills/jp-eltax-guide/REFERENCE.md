## eLTAXとは

eLTAX（エルタックス）は、地方税における手続きを、インターネットを利用して電子的に行うシステムです。地方税共同機構が運営しています。

### 利用可能な手続き

| カテゴリ | 手続き |
|----------|--------|
| 法人関係 | 法人住民税、法人事業税、事業所税 |
| 個人住民税 | 給与支払報告書、特別徴収関係届出 |
| 固定資産税 | 償却資産申告 |
| 納税 | 電子納付（全税目） |

### 利用時間

- 月曜日〜金曜日: 8:30〜24:00
- 土日祝日・年末年始: 利用不可

## Windows/Edge環境での設定

### 動作環境

- **OS**: Windows 10/11（日本語版、64bit推奨）
- **ブラウザ**: Microsoft Edge（最新版推奨）
- **電子証明書**:
  - 公的個人認証サービス（マイナンバーカード）
  - 商業登記電子証明書
  - 税理士用電子証明書
  - その他対応証明書

### PCdesk DL版のインストール

```
手順:
1. https://www.eltax.lta.go.jp/ にアクセス
2. 「PCdesk（DL版）」→「ダウンロード」をクリック
3. 最新版のインストーラをダウンロード
4. ダウンロードしたファイルを実行
5. インストールウィザードの指示に従う
6. インストール完了後、PCdeskを起動
7. 利用者IDと暗証番号でログイン
```

### Edgeの設定

#### ポップアップの許可

```
手順:
1. Edge 設定（右上の「...」→「設定」）
2. 「Cookie とサイトのアクセス許可」
3. 「ポップアップとリダイレクト」
4. 「許可」セクションに「https://www.eltax.lta.go.jp」を追加
5. 「https://www.portal.eltax.lta.go.jp」も追加
```

#### Cookieの設定

```
手順:
1. Edge 設定 → 「Cookie とサイトのアクセス許可」
2. 「Cookie とサイト データの管理と削除」
3. 「サイトに Cookie データの保存と読み取りを許可する」がオン
4. 「サードパーティの Cookie をブロックする」がオフ（推奨）
```

### 電子証明書の準備（Windows）

**対応証明書の種類**:

| 証明書 | 用途 | 取得方法 |
|--------|------|----------|
| 公的個人認証（マイナンバーカード） | 個人・法人代表者 | 市区町村窓口 |
| 商業登記電子証明書 | 法人 | 法務局 |
| 税理士用電子証明書 | 税理士 | 日税連 |

**ICカードリーダーの設定**:
```
手順:
1. ICカードリーダーをUSBで接続
2. ドライバをインストール（必要な場合）
3. JPKIクライアントソフトをインストール
4. カードを挿入して動作確認
```

### 給与支払報告書の作成（DL版）

```
手順:
1. PCdesk DL版を起動
2. メニューから「申告書作成」を選択
3. 「給与支払報告書」を選択
4. 提出先市区町村を選択
5. 給与データを入力
   - 手入力、またはCSVインポート
   - 給与ソフトからのデータ連携も可能
6. 入力内容を確認
7. 電子署名を付与
8. 送信
9. 送信結果を確認
```

### 法人住民税・事業税の申告（DL版）

```
手順:
1. PCdesk DL版を起動
2. メニューから「申告書作成」を選択
3. 該当する税目を選択
   - 法人都道府県民税
   - 法人市町村民税
   - 法人事業税
4. 申告データを作成
5. 電子署名を付与
6. 送信
7. 受付完了通知を確認
```

## macOS/Safari環境での設定

### 動作環境

- **OS**: macOS 10.12以上（日本語版）
- **ブラウザ**: Safari（最新版推奨）
- **電子証明書**: 公的個人認証サービスのみ対応

### Safariの設定

#### ポップアップブロックの解除

```
手順:
1. Safari メニュー → 設定（または環境設定）
2. 「Webサイト」タブをクリック
3. 左側メニューから「ポップアップウィンドウ」を選択
4. 右下の「以下のWebサイトでのみポップアップウィンドウを許可」で
   「https://www.portal.eltax.lta.go.jp」を「許可」に設定
```

#### Cookieの設定

```
手順:
1. Safari メニュー → 設定
2. 「プライバシー」タブをクリック
3. 「Cookieとwebサイトのデータ」で
   「すべてのCookieをブロック」がオフになっていることを確認
```

#### JavaScriptの有効化

```
手順:
1. Safari メニュー → 設定
2. 「セキュリティ」タブをクリック
3. 「JavaScriptを有効にする」にチェック
```

### 署名用プラグインのインストール（macOS）

```
手順:
1. eLTAX公式サイトにアクセス
2. 「パソコン環境の準備」→「署名用のプラグインをインストール」
3. 「ELPKISetup.dmg」をダウンロード
4. dmgファイルをダブルクリックして開く
5. インストーラの指示に従ってインストール
6. Safariを再起動
```

### 電子証明書の準備（macOS）

macOSで利用可能な電子証明書:
- **公的個人認証サービス（マイナンバーカード）**のみ

必要なもの:
- マイナンバーカード
- ICカードリーダー（macOS対応のもの）
- 利用者クライアントソフト

## 手続き別ガイド

### 利用届出（新規）- WEB版共通

```
手順:
1. https://www.eltax.lta.go.jp/ にアクセス
2. 「PCdesk（WEB版）」をクリック
3. 「利用届出（新規）」を選択
4. 利用規約を確認し同意
5. 利用届出の種類を選択（法人/個人）
6. 必要事項を入力
   - 法人の場合: 法人番号、法人名、所在地等
   - 個人の場合: 氏名、住所、生年月日等
7. 暗証番号を設定
8. 内容を確認して送信
9. 利用者IDと仮暗証番号を控える
```

### 電子納付

#### インターネットバンキング納付

```
手順:
1. PCdesk（WEB版またはDL版）にログイン
2. 「納付情報発行依頼」を選択
3. 納付する税目・金額を確認
4. 「納付情報発行」をクリック
5. 納付番号・確認番号を控える
6. 金融機関のインターネットバンキングにログイン
7. 「ペイジー」または「収納サービス」を選択
8. 納付番号等を入力して納付
```

#### クレジットカード納付

```
手順:
1. PCdesk（WEB版またはDL版）にログイン
2. 「納付情報発行依頼」を選択
3. 納付する税目・金額を確認
4. 「クレジットカード納付」を選択
5. カード情報を入力
6. 決済完了を確認
```

**注意**: クレジットカード納付には手数料がかかります

## トラブルシューティング

### ログインできない

```
確認事項:
1. 利用者IDと暗証番号が正しいか
2. 仮暗証番号の場合は初回ログイン時に変更が必要
3. 利用時間内か（月〜金 8:30〜24:00）
4. ブラウザのCookieが有効か
```

### 電子署名ができない

```
確認事項:
1. 署名用プラグインがインストールされているか
2. ICカードリーダーが正しく接続されているか
3. 電子証明書の暗証番号が正しいか
4. 電子証明書の有効期限が切れていないか
5. JPKIクライアントソフトが正しくインストールされているか
```

### ポップアップがブロックされる

**Windows/Edge**:
```
対処法:
1. Edge 設定 → Cookie とサイトのアクセス許可
2. ポップアップとリダイレクト → 許可に追加
3. Edgeを再起動
```

**macOS/Safari**:
```
対処法:
1. Safari 設定 → Webサイト → ポップアップウィンドウ
2. portal.eltax.lta.go.jp を「許可」に設定
3. Safariを再起動
```

### 画面が正しく表示されない

```
対処法:
1. ブラウザのキャッシュをクリア
2. ブラウザを再起動
3. 別のブラウザで試す（Edge推奨）
```

### DL版が起動しない（Windows）

```
確認事項:
1. Windowsが最新の状態か
2. .NET Frameworkがインストールされているか
3. セキュリティソフトがブロックしていないか
4. 管理者権限で実行を試す
```
