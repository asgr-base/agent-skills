# e-Tax（国税電子申告・納税システム）ガイド

## 重要: 最新情報の取得

**このスキルで回答する際は、必ずWebSearchで公式サイトから最新情報を取得してから回答すること。**

```
必須ワークフロー:
1. ユーザーの質問内容を特定
2. WebSearchで「e-Tax [手続き名] [キーワード]」を検索
3. 取得した最新情報に基づいて回答
4. 情報源のURLを明記
```

| 機能 | Windows（インストール版） | Windows（WEB版） | macOS（WEB版） |

| 確定申告書作成 | ○ | ○ | ○ |
| 法人税申告 | ○ | ○ | ○ |

| 消費税申告 | ○ | ○ | ○ |

**推奨環境**:
- **個人の確定申告**: 確定申告書等作成コーナー（WEB版）
- **法人・頻繁な申告**: e-Taxソフト（インストール版）
- **macOSユーザー**: e-Taxソフト（WEB版）

## ソフトウェアの種類

### 1. 確定申告書等作成コーナー（WEB版）

個人の確定申告に最適。ブラウザのみで完結。

```
アクセス:
https://www.keisan.nta.go.jp/kyoutu/ky/sm/top
```

### 2. e-Taxソフト（WEB版）

ブラウザで各種申告・届出が可能。

```
アクセス:
https://clientweb.e-tax.nta.go.jp/
```

### 3. e-Taxソフト（インストール版）

Windows専用。法人や頻繁に申告を行う場合に推奨。

```
ダウンロード:
https://www.e-tax.nta.go.jp/download/e-taxSoftDownLoad.htm
```

## Windows環境

### e-Taxソフト（インストール版）のセットアップ

```
手順:
1. https://www.e-tax.nta.go.jp/ にアクセス
2. 「e-Taxソフトのダウンロード」を選択
3. インストーラを実行
4. 利用者識別番号でログイン
```

### 動作環境

- **OS**: Windows 10/11（日本語版）
- **ブラウザ**: Microsoft Edge、Google Chrome（最新版推奨）
- **電子証明書**:
  - マイナンバーカード（公的個人認証サービス）
  - 商業登記電子証明書
  - 税理士用電子証明書

### 主な手続き（Windows）

**確定申告書の作成・送信**:
```
手順:
1. 確定申告書等作成コーナーにアクセス
2. 「作成開始」→申告書の種類を選択
3. 収入・控除情報を入力
4. マイナンバーカードで電子署名
5. 送信
```

**法人税申告**:
```
手順:
1. e-Taxソフトを起動
2. 「申告・申請等」→「法人税」を選択
3. 申告データを作成（または会計ソフトから取込）
4. 電子署名を付与
5. 送信
```

**源泉所得税の納付**:
```
手順:
1. e-Taxソフト（WEB版）にログイン
2. 「申告・申請等」→「源泉所得税」
3. 納付情報を入力
4. 電子納税または納付書出力
```

## macOS環境

### 対応状況

macOSではWEB版のみ利用可能です。

| ソフトウェア | macOS対応 | 備考 |
|-------------|-----------|------|
| 確定申告書等作成コーナー | ○ | Safari/Chrome対応 |
| e-Taxソフト（WEB版） | ○ | Safari/Chrome対応 |
| e-Taxソフト（インストール版） | × | Windows専用 |

### Safari設定

```
手順:
1. Safari メニュー → 設定
2. 「Webサイト」タブ → 「ポップアップウィンドウ」
3. e-tax.nta.go.jp を「許可」に設定
4. 「プライバシー」→「サイト越えトラッキングを防ぐ」のチェックを外す
```

### マイナンバーカードの読み取り

```
必要なもの:
- ICカードリーダー（macOS対応のもの）
- マイナンバーカード
- 利用者クライアントソフト（JPKI利用者ソフト）

セットアップ:
1. JPKI利用者ソフトをインストール
   https://www.jpki.go.jp/download/
2. ICカードリーダーを接続
3. マイナンバーカードをセット
4. 動作確認
```

## 共通手続き

### 利用開始届出

```
手順:
1. https://www.e-tax.nta.go.jp/ にアクセス
2. 「e-Taxの開始届出」をクリック
3. 「個人の方」または「法人の方」を選択
4. 必要事項を入力
5. 利用者識別番号と暗証番号を取得
```

### マイナポータル連携

マイナポータルと連携すると、医療費や保険料のデータを自動取得できます。

```
手順:
1. マイナポータルにログイン
2. 「もっとつながる」→「e-Tax」を連携
3. 確定申告書等作成コーナーで「マイナポータル連携」を選択
```

### 電子納税

- **ダイレクト納付**: 届出した預貯金口座から即時引落
- **インターネットバンキング**: Pay-easy（ペイジー）対応
- **クレジットカード**: 「国税クレジットカードお支払サイト」経由
- **スマホアプリ納付**: PayPay、d払い、au PAY等

## よくある質問

### Q: e-TaxとeLTAXの違いは？

A: e-Taxは**国税**（所得税、法人税、消費税等）、eLTAXは**地方税**（住民税、事業税等）のシステムです。

### Q: 利用時間は？

A: 通常期間: 月〜金曜日 8:30〜24:00
   確定申告期間（2月16日〜3月15日頃）: 24時間

### Q: マイナンバーカードがない場合は？

A: ID・パスワード方式で申告可能（税務署で事前に届出が必要）。

### Q: 過去の申告データを確認したい

A: メッセージボックスで送信済みデータを確認できます（保存期間あり）。
