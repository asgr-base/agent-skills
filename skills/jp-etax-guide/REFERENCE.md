## Windows環境での設定

### e-Taxソフト（インストール版）のインストール

```
システム要件:
- OS: Windows 10/11（日本語版、32bit/64bit）
- メモリ: 2GB以上推奨
- ディスク: 500MB以上の空き容量
- 画面解像度: 1024×768以上
```

```
インストール手順:
1. https://www.e-tax.nta.go.jp/download/e-taxSoftDownLoad.htm にアクセス
2. 「e-Taxソフトのダウンロード」をクリック
3. ダウンロードしたファイルを実行
4. 画面の指示に従ってインストール
5. 初回起動時に利用者ファイルを作成
```

### ブラウザ設定（Edge/Chrome）

```
Edge設定:
1. 設定 → Cookie とサイトのアクセス許可
2. ポップアップとリダイレクト → 許可に以下を追加:
   - e-tax.nta.go.jp
   - keisan.nta.go.jp
3. JavaScript → 有効を確認
```

```
Chrome設定:
1. 設定 → プライバシーとセキュリティ → サイトの設定
2. ポップアップとリダイレクト → 許可に以下を追加:
   - e-tax.nta.go.jp
   - keisan.nta.go.jp
3. JavaScript → 許可
```

### ICカードリーダーの設定

```
対応ICカードリーダー:
- 接触型（USB接続）: SCR3310-NTTCom、ACR39U等
- 非接触型: PaSoRi（RC-S380等）
- 複合機内蔵型

ドライバインストール:
1. ICカードリーダーを接続
2. 製品付属のドライバをインストール（または自動認識）
3. JPKI利用者ソフトで動作確認
```

## macOS環境での設定

### 対応ブラウザ

| ブラウザ | 確定申告書等作成コーナー | e-Taxソフト（WEB版） |
|---------|------------------------|---------------------|
| Safari | ○（推奨） | ○ |
| Chrome | ○ | ○ |
| Firefox | △（一部機能制限） | △ |

### Safari詳細設定

```
ポップアップ許可:
1. Safari → 設定 → Webサイト
2. 「ポップアップウィンドウ」を選択
3. 以下のサイトを「許可」に設定:
   - e-tax.nta.go.jp
   - keisan.nta.go.jp
   - clientweb.e-tax.nta.go.jp
```

```
プライバシー設定:
1. Safari → 設定 → プライバシー
2. 「サイト越えトラッキングを防ぐ」のチェックを外す
   （e-Tax利用中のみ推奨）
3. Cookieを「常に許可」または「訪問したWebサイトは許可」
```

### JPKI利用者ソフトのインストール

```
ダウンロード:
https://www.jpki.go.jp/download/mac.html

インストール手順:
1. dmgファイルをダウンロード
2. ダウンロードしたファイルを開く
3. JPKIUtility.appをアプリケーションフォルダにドラッグ
4. 初回起動時にセキュリティ許可を付与
```

### macOS対応ICカードリーダー

```
確認済み対応機種:
- Sony PaSoRi RC-S380（非接触型）
- Gemalto/Thales IDPrime（接触型）
- ACR39U-N1（接触型）

注意: 製品によってはmacOS用ドライバが必要
```

## 電子証明書の準備

### マイナンバーカード（公的個人認証サービス）

```
取得手順:
1. 市区町村役場で申請（または郵送/オンライン申請）
2. 交付通知書が届いたら受け取り
3. 署名用電子証明書の暗証番号（6〜16桁英数字）を設定
4. 利用者証明用電子証明書の暗証番号（4桁数字）を設定
```

```
有効期限:
- 署名用電子証明書: 5年（住所変更で失効）
- 利用者証明用電子証明書: 5年
- カード自体: 10年（成人）/5年（未成年）
```

### 商業登記電子証明書

```
取得手順:
1. 法務省オンライン申請システムで申請
2. 手数料納付（証明期間による）
3. ファイル形式で証明書を取得
4. e-Taxソフトにインポート
```

```
手数料（2024年現在）:
- 証明期間3ヶ月: 1,300円
- 証明期間6ヶ月: 2,300円
- 証明期間12ヶ月: 4,300円
- 証明期間24ヶ月: 8,300円
- 証明期間27ヶ月: 9,300円
```

## 各種申告手続きの詳細

### 所得税の確定申告

```
申告期間: 2月16日〜3月15日（還付申告は1月1日から）

確定申告書等作成コーナーでの手順:
1. https://www.keisan.nta.go.jp/ にアクセス
2. 「作成開始」をクリック
3. 提出方法を選択（e-Tax/印刷）
4. 申告書の種類を選択
   - 所得税: 給与・年金・事業・不動産等
   - 青色申告決算書/収支内訳書
5. 収入金額を入力
6. 所得控除を入力（医療費、社会保険料等）
7. 税額控除を入力（住宅ローン控除等）
8. 電子署名（マイナンバーカード）
9. 送信
```

### 法人税の申告

```
申告期限: 事業年度終了後2ヶ月以内

e-Taxソフト（インストール版）での手順:
1. 利用者ファイルを選択
2. 「申告・申請等」→「新規作成」
3. 「法人税」→該当する様式を選択
4. 別表一から順に入力
5. 決算書データを添付（XBRL形式推奨）
6. 電子署名を付与
7. 送信
```

### 源泉所得税の納付

```
納付期限:
- 原則: 翌月10日
- 納期の特例: 7月10日、1月20日（届出が必要）

e-Taxソフト（WEB版）での手順:
1. https://clientweb.e-tax.nta.go.jp/ にログイン
2. 「申告・申請等」→「新規作成」
3. 「源泉所得税及復興特別所得税」を選択
4. 納付書情報を入力
5. 送信後、電子納税
```

### 消費税の申告

```
申告期限:
- 個人事業主: 3月31日
- 法人: 事業年度終了後2ヶ月以内

申告手順:
1. 課税売上・課税仕入を集計
2. 確定申告書等作成コーナーまたはe-Taxソフトで入力
3. 電子署名を付与
4. 送信
```

## 電子納税の詳細

### ダイレクト納付

```
事前届出:
1. 「ダイレクト納付利用届出書」を提出
2. 届出から約1ヶ月で利用可能

納付手順:
1. 申告書を送信
2. 「今すぐ納付」または「期日指定納付」を選択
3. 届出した口座から引落
```

### インターネットバンキング（ペイジー）

```
対応金融機関:
- 都市銀行、地方銀行、信用金庫、ゆうちょ銀行等
- 事前にインターネットバンキング契約が必要

納付手順:
1. 申告後に「納付情報登録」
2. 収納機関番号・納付番号等を取得
3. 各金融機関のインターネットバンキングで納付
```

### クレジットカード納付

```
対応ブランド: Visa、Mastercard、JCB、American Express、Diners Club

手数料:
- 1万円まで: 83円（税込）
- 以降1万円ごとに加算

納付手順:
1. 「国税クレジットカードお支払サイト」にアクセス
2. 納付情報を入力
3. カード情報を入力
4. 納付実行
```

### スマホアプリ納付

```
対応アプリ:
- PayPay
- d払い
- au PAY
- LINE Pay
- メルペイ
- Amazon Pay

納付上限: 30万円まで

手順:
1. 国税スマートフォン決済専用サイトにアクセス
2. 納付情報を入力
3. アプリを選択して納付
```

## トラブルシューティング

### 共通の問題

| 症状 | 原因 | 対処法 |
|------|------|--------|
| ログインできない | 利用者識別番号・暗証番号の誤り | 入力内容を確認、暗証番号再設定 |
| 電子署名エラー | 証明書の有効期限切れ | 証明書を更新 |
| 送信エラー | サーバー混雑 | 時間をおいて再送信 |
| ポップアップがブロック | ブラウザ設定 | ポップアップを許可 |

### Windows固有の問題

| 症状 | 原因 | 対処法 |
|------|------|--------|
| e-Taxソフトが起動しない | Javaの問題 | e-Taxソフトを再インストール |
| ICカードリーダー認識しない | ドライバの問題 | ドライバを再インストール |
| 「予期しないエラー」 | キャッシュの問題 | ブラウザキャッシュをクリア |

### macOS固有の問題

| 症状 | 原因 | 対処法 |
|------|------|--------|
| JPKI利用者ソフトが開けない | セキュリティ設定 | システム設定でアプリを許可 |
| カードが読み取れない | リーダーの互換性 | macOS対応リーダーを使用 |
| Safari で動作しない | 拡張機能の競合 | 拡張機能を無効化 |

### エラーコード一覧

| コード | 内容 | 対処法 |
|--------|------|--------|
| HJS0407E | 電子署名エラー | ICカードリーダー接続確認、証明書確認 |
| HJS0430E | 通信エラー | ネットワーク接続確認、再送信 |
| CC01 | 利用者識別番号エラー | 番号を確認、再入力 |
| CC04 | 暗証番号ロック | 税務署で解除手続き |
