# マネーフォワード クラウド ひとり法人プラン ガイド

## 重要: 最新情報の取得

**このスキルで回答する際は、必ずWebSearchまたはWebFetchで公式サポートサイトから最新情報を取得してから回答すること。**

本ドキュメントの情報は参考用であり、料金・機能・制限事項は変更される可能性があります。

```
必須ワークフロー:
1. ユーザーの質問内容を特定
2. WebSearchで最新情報を検索（または該当サポートページをWebFetch）
3. 取得した最新情報に基づいて回答
4. 情報源のURLを明記
```

## 概要

ひとり法人プランは、経営者1名のみの法人向けに最適化されたマネーフォワード クラウドのプランです。

**料金**: 年払い 2,480円/月（29,760円/年）、月払い 3,980円/月

**対象**: 従業員がおらず取引数が多くない「ひとり社長」や資産管理会社

## 含まれるサービス（全12サービス）

| カテゴリ | サービス |
|---------|----------|
| 経理財務 | 会計、請求書、経費、債務支払 |
| 人事労務 | 給与、勤怠、年末調整、社会保険、マイナンバー、人事管理 |
| その他 | 契約、Box |

## 主な制限事項

| 項目 | ひとり法人 | スモールビジネス | ビジネス |
|------|-----------|-----------------|----------|
| 利用人数 | 1名 | 3名 | 従量課金 |
| 年間仕訳数 | 500件 | 無制限 | 無制限 |
| AI-OCR | 30件/月 | 60件/月 | 従量課金 |
| 部門登録 | 2部門 | 2部門 | 無制限 |
| 消費税申告書 | 不可 | 可 | 可 |
| FBデータ作成 | 不可 | 可 | 可 |

## クイックスタート

### 1. 会計の初期設定

```
手順:
1. 事業者設定（会社名、決算期など）
2. 金融機関連携（銀行、クレジットカード）
3. 勘定科目の確認・カスタマイズ
4. 開始残高の入力
```

**サポートリンク**: [使い方ガイド](https://biz.moneyforward.com/support/account/guide)

### 2. 給与計算（ひとり社長）

ひとり社長でも自分を従業員として登録し給与計算を行います。

```
手順:
1. 従業員情報に自分を登録
2. 締め日グループを設定
3. 給与明細を作成・確定
4. 会計への自動連携を確認
```

**サポートリンク**: [給与使い方ガイド](https://biz.moneyforward.com/support/payroll/guide)

### 3. 年末調整

年末調整機能で扶養控除等申告書を自動作成できます。

```
手順:
1. 10月中に初期設定を完了
2. 1月〜12月の給与明細を全て確定
3. 手続きを作成し、申告書を回収
4. 控除額・税額を自動計算
```

**サポートリンク**: [年末調整の流れ](https://biz.moneyforward.com/support/tax-adjustment/guide/flow/managements-flow01.html)

### 4. 社会保険

協会けんぽの場合、都道府県を設定するだけで料率が自動設定されます。

**サポートリンク**: [社会保険サポート](https://biz.moneyforward.com/support/social-insurance/)

## よくある質問

### Q: 仕訳数500件を超えそうな場合は？

A: スモールビジネスプラン（4,480円/月）へのアップグレードが必要です。契約期間中のプラン変更はできないため、更新時に変更してください。

### Q: 消費税申告書を作成したい

A: ひとり法人プランでは消費税申告書の作成機能は利用できません。スモールビジネスプラン以上が必要です。

### Q: 電話サポートはある？

A: 会員登録から2ヶ月間、事前予約制（10:00〜17:00）で電話サポートを利用できます。
