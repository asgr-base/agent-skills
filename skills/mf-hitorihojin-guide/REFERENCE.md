## 料金・プラン詳細

### ひとり法人プラン

| 項目 | 内容 |
|------|------|
| 年払い | 2,480円/月（29,760円/年） |
| 月払い | 3,980円/月 |
| 初期費用 | 0円 |
| 無料トライアル | 1ヶ月 |
| 利用条件 | 管理コンソールのユーザーが1名のみ（士業除く） |

### プラン変更に関する注意

- 契約期間中のプラン変更は**不可**
- ユーザー数の拡張は**不可**
- 更新時にプラン変更を検討すること
- 人数が増える可能性がある場合はビジネスプランを推奨

### クラウド給与

#### ひとり社長の給与計算フロー

```
1. 従業員情報登録
   - 自分を従業員として登録
   - 基本情報、給与情報を入力
   - 口座情報は任意（FBデータ作成不可のため）

2. 締め日グループ設定
   - 締め日、支給日を設定
   - 社会保険料の徴収方法を選択

3. 毎月の給与計算
   - 給与明細を作成
   - 内容を確認して確定
   - 会計への自動連携

4. 年末調整への連携
   - 1月〜12月の給与を全て確定
   - 年末調整機能で申告書を自動作成
```

#### 社会保険の設定

```
協会けんぽの場合:
- 健康保険の種類: 協会管掌事業所
- 管轄: 本店所在地の都道府県
→ 健康保険・厚生年金の料率が自動設定

健康保険組合の場合:
- 健康保険の種類: 組合管掌事業所
- 料率を手動設定
```

### クラウド年末調整

#### 年末調整スケジュール

| 時期 | 作業内容 |
|------|----------|
| 10月 | 初期設定完了 |
| 11月上旬 | 従業員への申告依頼開始 |
| 11月中旬 | 申告書の回収・確認 |
| 12月上旬 | 年末調整計算の実行 |
| 12月中旬 | 源泉徴収票の発行 |
| 1月 | 法定調書の提出 |

#### 管理者向けフロー

```
1. 初期設定
   - 申告事業所の設定
   - 給与データの連携確認
   - 手続きの作成

2. 申告書の回収
   - 従業員（自分）への依頼送信
   - 扶養控除等申告書の入力
   - 保険料控除申告書の入力

3. 計算・確認
   - 控除額の自動計算
   - 過不足税額の確認
   - 源泉徴収票の確認

4. 書類出力
   - 源泉徴収票の発行
   - 給与支払報告書の作成
```

## トラブルシューティング

### 仕訳数が上限に近い

```
対処法:
1. 仕訳登録数を確認（設定 > 事業者情報）
2. 不要な仕訳を削除
3. 仕訳ルールで効率化
4. 500件超える場合はプランアップグレードを検討
```

### 金融機関連携がエラーになる

```
対処法:
1. 金融機関のメンテナンス情報を確認
2. 連携を一度解除して再設定
3. 金融機関側のパスワード変更を確認
```

### 年末調整で給与が反映されない

```
対処法:
1. 1月〜12月の給与明細が全て「確定」済みか確認
2. 年の途中からクラウド給与を使い始めた場合は
   使用開始前の給与明細も作成が必要
3. クラウド給与とクラウド年末調整の連携設定を確認
```

### 会計と給与の連携がうまくいかない

```
対処法:
1. 連携設定を確認（給与 > 設定 > 連携サービス）
2. 給与明細が「確定」済みか確認
3. 勘定科目のマッピングを確認
```
