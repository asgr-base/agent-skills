# AML/CFTスタンダードコース試験対策ガイド

## 試験概要

### 基本情報
- **主催**: 一般社団法人 金融財政事情研究会
- **公式サイト**: https://www.kinzai.or.jp/kentei/5h7.html
- **試験方式**: CBT（Computer Based Testing）
- **受験資格**: 特になし（誰でも受験可能）

### 試験形式（推定）
※公式サイトで最新情報を確認してください

- **問題数**: 約50問（多岐選択式）
- **試験時間**: 90分程度
- **合格基準**: 70%以上の正答率（推定）
- **出題範囲**: AML/CFTの基礎から実務まで幅広く

## シラバス（出題範囲）

### 1. AML/CFTの基礎知識（20%）
- マネーローンダリングの定義・手法
- テロ資金供与の特徴
- FATF勧告の概要
- 国際的な取組み

### 2. 法令・規制（25%）
- 犯罪収益移転防止法
- 組織的犯罪処罰法
- 外為法（制裁関連）
- 金融庁ガイドライン

### 3. 顧客管理（25%）
- 本人確認（KYC）の方法
- 実質的支配者の確認
- 継続的顧客管理（CDD）
- リスクベースアプローチ

### 4. 取引モニタリング・報告（20%）
- 疑わしい取引の検知
- JAFIC への届出
- 高額現金取引の届出
- 外国送金等の記録保存

### 5. 体制整備・実務（10%）
- 経営陣の関与
- 研修・教育
- 内部監査
- システム対応

## 学習戦略

### 短期集中型の学習計画

#### Phase 1: 基礎固め（1週間）
- 「マネロン・テロ資金供与対策キーワード100」を通読
- 各キーワードの定義を理解
- 重要用語をノートにまとめる

#### Phase 2: 法令・実務（1週間）
- 犯罪収益移転防止法の条文を確認
- 金融庁ガイドラインを読む
- eKYCの実務との関連を整理

#### Phase 3: 演習・復習（1週間）
- 演習問題を解く（後述のサンプル問題など）
- 間違えた問題を重点的に復習
- 重要トピックの再確認

### 重点学習トピック

**優先度：高**
1. 犯罪収益移転防止法の本人確認方法（4つの方法、eKYC含む）
2. リスクベースアプローチの考え方
3. 疑わしい取引の判断基準・報告義務
4. FATF勧告の主要内容

**優先度：中**
5. マネーロンダリングの3段階プロセス
6. 実質的支配者の確認
7. 制裁リストスクリーニング
8. 高額現金取引の届出基準

**優先度：低**
9. 国際的な組織・枠組み
10. 歴史的な経緯

## 出題傾向（推定）

### 頻出テーマ
- 本人確認の方法と要件
- 疑わしい取引の具体例
- リスク評価の考え方
- 法令の条文解釈

### 出題形式の特徴
- **定義問題**: 用語の正確な定義を問う
- **事例問題**: 具体的な取引事例でAML/CFT対応を問う
- **法令問題**: 条文の内容や要件を問う
- **判断問題**: リスク評価や疑わしい取引の判断を問う

## サンプル問題

### 問題1: 基礎知識
```
マネーローンダリングの3段階プロセスのうち、不正資金を金融システムに投入する段階を何と呼ぶか？

A. インテグレーション
B. レイヤリング
C. プレースメント
D. スマーフィング

正解: C
```

### 問題2: 法令
```
犯罪収益移転防止法における本人確認で、eKYC（オンライン完結）の方法として認められているものはどれか？

A. 運転免許証のコピーを郵送
B. 身分証ICチップ読み取り＋顔写真撮影
C. 電子メールでの本人確認
D. 電話での口頭確認

正解: B
```

### 問題3: 実務
```
次のうち、疑わしい取引として届出を検討すべきものはどれか？

A. 給与振込口座への定期的な入金
B. 公共料金の口座引落し
C. 取引目的の説明がつかない多額の現金入金
D. 住宅ローンの返済

正解: C
```

### 問題4: リスク評価
```
リスクベースアプローチにおいて、高リスク顧客として追加的な確認（EDD）が必要となるのはどれか？

A. 国内在住の会社員
B. FATF高リスク国の国籍を持つ顧客
C. 地方公共団体
D. 上場企業の従業員

正解: B
```

## 学習のコツ

1. **体系的に整理**: トピックごとにマインドマップ作成
2. **実務と関連付け**: eKYCプロダクトでの具体例を考える

### 間違えやすいポイント

- 「疑わしい取引」と「高額現金取引」の届出の違い

- JAFICとFATFの役割の違い
