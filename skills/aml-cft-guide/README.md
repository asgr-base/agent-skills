# AML/CFT Guide Skill

AML/CFT（マネーローンダリング・テロ資金供与対策）の専門知識を提供し、金融財政事情研究会のAML/CFTスタンダードコース試験対策を支援するClaude Code Agent Skillです。

## 概要

このスキルは以下を提供します：

1. **専門知識の提供**
   - マネーローンダリング・テロ資金供与対策の基礎知識
   - 信頼性の高い情報源（金融庁、JAFIC、FATF等）からの最新情報
   - eKYCプロダクトへの実務適用の考察

2. **試験対策支援**
   - AML/CFTスタンダードコースのシラバス解説
   - 出題傾向分析と重要ポイントの整理
   - 学習計画の助言と演習問題の作成

## ファイル構成

```
aml-cft-guide/

├── RESOURCES.md      # 信頼できる情報源リスト
└── README.md         # このファイル
```

## 使い方

### 基本的な質問

**例**: 「FATFとは何ですか？」

Claudeが自動的にこのスキルを起動し、以下を実行：
1. FATF（金融活動作業部会）の基本情報を説明
2. 必要に応じてWebSearchで最新情報を取得
3. Sources:セクションで情報源を明示

### 試験対策

**例**: 「AML/CFTスタンダードコースの勉強方法を教えてください」

Claudeが以下を実行：
1. EXAM-GUIDE.mdを参照し、試験概要とシラバスを提示
2. WebSearchで最新の試験日程・申込情報を取得
3. 学習計画を提案（Phase 1-3の3週間プラン）
4. 重要トピックと演習問題を提供

### 実務適用

**例**: 「eKYCで実質的支配者をどう確認すべきか？」

Claudeが以下を実行：
1. TOPICS.mdで実質的支配者の定義を確認
2. WebFetchで犯罪収益移転防止法施行規則の最新要件を取得
3. eKYCシステムでの実装上の考慮点を提示

## 情報源の優先順位

1. **公式機関**: 金融庁、JAFIC、FATF
2. **業界団体**: 金融財政事情研究会、全国銀行協会
3. **専門出版**: EY、PwC、Deloitte、KPMG等の監査法人
4. **学習教材**: 「マネロン・テロ資金供与対策キーワード100【第3版】」

## 主要トピック

- AML基礎（マネーローンダリングの3段階プロセス）
- CFT基礎（テロ資金供与の特徴）

## インストール

このスキルは既に `.claude/skills/aml-cft-guide/` に配置されています。

Claude Codeが自動的にスキルを読み込み、AML/CFT関連の質問時に起動します。

## トリガーキーワード

以下のキーワードを含む質問で自動起動：
- AML, CFT
- マネロン, マネーロンダリング
- テロ資金供与
- KYC, eKYC
- 疑わしい取引
- リスクベースアプローチ
- 犯罪収益移転防止法
- FATF

## 注意事項

1. **法令情報の正確性**: 法律・規制は変更される可能性があるため、必ず最新の公式情報を確認してください
2. **個別法的助言の禁止**: 一般的な情報提供にとどめ、個別案件への法的助言は行いません
3. **試験問題の著作権**: 実際の過去問は提供せず、オリジナルの演習問題のみ作成します
4. **情報源の明示**: 回答には必ずソース（URL、文献名）を記載します
