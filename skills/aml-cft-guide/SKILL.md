# AML/CFT専門知識・試験対策ガイド

## 概要

このスキルは以下を提供：
1. **AML/CFT専門知識**: マネーローンダリング・テロ資金供与対策の信頼性ある情報
2. **試験対策**: 金融財政事情研究会「AML/CFTスタンダードコース」の学習支援

## 主要機能

### 1. 専門知識の提供

以下のトピックに関する正確な情報を提供：
- **AML基礎**: マネーローンダリングの手法、3段階プロセス（プレースメント・レイヤリング・インテグレーション）
- **CFT基礎**: テロ資金供与の特徴、国際的な取組み
- **リスクベースアプローチ**: リスク評価と低減措置
- **顧客確認（KYC）**: 本人確認、実質的支配者の確認、eKYC
- **取引モニタリング**: 疑わしい取引の検知・報告
- **規制・法令**: 犯罪収益移転防止法、外為法、FATF勧告

詳細トピックは [TOPICS.md](TOPICS.md) を参照。

### 2. 試験対策支援

**試験情報**:
- 主催: 一般社団法人 金融財政事情研究会
- 公式URL: https://www.kinzai.or.jp/kentei/5h7.html
- 試験形式: CBT（Computer Based Testing）
- 出題範囲: AML/CFTの基礎から実務まで幅広く

**提供する支援**:
- シラバス解説
- 出題傾向分析
- 重要ポイントの整理
- 学習計画の助言
- 模擬問題の作成（過去問ではなく、シラバスに基づく演習問題）

試験対策の詳細は [EXAM-GUIDE.md](EXAM-GUIDE.md) を参照。

## 情報源の優先順位

信頼性の高い順に情報を取得：

1. **公式機関**:
   - 金融庁: https://www.fsa.go.jp/
   - JAFIC（警察庁 犯罪収益移転防止対策室）: https://www.npa.go.jp/sosikihanzai/jafic/
   - FATF（金融活動作業部会）: https://www.fatf-gafi.org/

2. **業界団体・資格機関**:
   - 金融財政事情研究会: https://www.kinzai.or.jp/
   - 全国銀行協会: https://www.zenginkyo.or.jp/

3. **専門出版・コンサル**:
   - EY、PwC、Deloitte、KPMG等の大手監査法人の公開資料

4. **学習教材**:
   - 「マネロン・テロ資金供与対策キーワード100【第3版】」（EYストラテジー・アンド・コンサルティング）

## 使用方法

### 基本的な質問対応

**ユーザーがAML/CFT関連の質問をした場合**:
1. まず既知の知識で回答できるか判断
2. 最新情報や正確な法令情報が必要な場合は `WebSearch` または `WebFetch` を使用
3. 情報源の優先順位に従い、信頼性の高いソースから取得
4. 回答には必ずソースを明記（Sources:セクション）

**例**:
```
ユーザー: 「実質的支配者の確認」とは何ですか？

対応:
1. 既知の知識で基本概念を説明
2. 金融庁や犯罪収益移転防止法の最新ガイドラインをWebFetchで確認
3. 回答にソースURLを含める
```

### 試験対策の支援

**ユーザーが試験対策を求めた場合**:
1. [EXAM-GUIDE.md](EXAM-GUIDE.md) を参照し、試験概要を確認
2. WebSearchで最新の試験情報（申込期間、試験日程など）を取得
3. シラバスに基づく学習計画を提案
4. 重要トピックの要約や演習問題を提供

**例**:
```
ユーザー: AML/CFTスタンダードコースの勉強方法を教えてください

対応:
1. EXAM-GUIDE.mdで試験範囲を確認
2. 金融財政事情研究会の公式サイトで最新情報を取得
3. トピック別の学習優先度を提案
4. 演習問題作成の支援
```

### eKYC実務への適用

**ユーザーがeKYCプロダクトとの関連を質問した場合**:
1. KYC/eKYCのトピック（TOPICS.md）を参照
2. 犯罪収益移転防止法の本人確認方法（オンライン完結型）の要件を説明
3. リスクベースアプローチの観点から実装時の考慮点を提示

## 重要な注意事項

1. **法令情報の正確性**: 法律や規制は変更される可能性があるため、必ず最新の公式情報を確認
2. **個別具体的な法的助言の禁止**: 一般的な情報提供にとどめ、個別案件への法的助言は行わない
3. **試験問題の著作権**: 実際の過去問は著作権保護されているため、オリジナルの演習問題のみ作成
4. **情報源の明示**: 回答には必ずソース（URL、文献名）を記載

## 使用例

### 例1: 基本概念の質問
```
ユーザー: FATFとは何ですか？

対応:
- FATFの正式名称、設立経緯、役割を説明
- FATF勧告の重要性を解説
- 日本におけるFATF勧告の実施状況に言及
- Sources: https://www.fatf-gafi.org/
```

### 例2: 試験対策
```
ユーザー: リスクベースアプローチの出題ポイントは？

対応:
- EXAM-GUIDE.mdでシラバスを確認
- リスク評価の手順、高リスク取引の判断基準などの重要ポイントを整理
- 演習問題を3問作成
- Sources: 金融財政事情研究会、金融庁ガイドライン
```

### 例3: 実務適用
```
ユーザー: eKYCで実質的支配者をどう確認すべきか？

対応:
- 犯罪収益移転防止法施行規則のオンライン完結型の要件を確認（WebFetch）
- 実質的支配者の定義と確認書類を説明
- eKYCシステムでの実装上の考慮点を提示
- Sources: 金融庁、JAFIC
```
