# ASGR Agent Skills

このリポジトリはClaude Code用のAgent Skillsコレクションです。

## インストール

```bash
/plugin marketplace add asgr-base/agent-skills
```

## 利用可能なスキル

### jp-law-verification

e-Gov法令APIを使用して日本の法令を検索・確認するスキル。

**使用例**:
- 「所得税法第183条を確認して」
- 「源泉徴収のタイミングについて法的根拠を教えて」

**依存関係**: e-gov-law MCPサーバーが必要

## スキルの場所

すべてのスキルは `skills/` ディレクトリに格納されています。
